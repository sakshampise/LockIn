# LockIn

A modern productivity web application designed to help users stay focused, organize their work, and build better productivity habits.

> 🚧 **Status:** Active Development

## Current Features

* 🔐 Secure authentication with Supabase
* ☁️ Cloud-backed data storage
* 🎯 Customizable focus sessions
* 🌙 Modern dark interface
* ⚡ Fast React + Vite frontend

## Planned Features

* ✅ Task management
* 📝 Notes & pages
* 📊 Productivity dashboard
* 📈 Focus analytics
* 🔍 Search
* ⚙️ Advanced focus presets & break scheduling

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* Framer Motion
* Supabase

## Getting Started

### Clone the repository

```bash
git clone https://github.com/sakshampise/LockIn.git
cd LockIn
```

### Install dependencies

```bash
npm install
```

### Configure environment variables

Create a `.env.local` file:

```env
VITE_SUPABASE_URL=your_project_url
VITE_SUPABASE_ANON_KEY=your_anon_key
```

### Run the development server

```bash
npm run dev
```
